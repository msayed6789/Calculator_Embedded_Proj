# Calculator_Embedded_Proj
It's a basic calculator with essential mathematical operations like addition, subtraction, multiplication, and division.
<br>
<br>

# Technical Features:
## It consists of AVR (atmega32), LCD and Keypad
## It performs fundamental mathematical operations such as addition, subtraction, multiplication, and division.
